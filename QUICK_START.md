# ⚡ Quick Start Guide

## 🎯 Get Running in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Visit: **http://localhost:5173/**

## 🛠 Customization

### Update Your Name
Edit: `client/src/components/HeroSection.tsx`

### Change Profile Photo
Edit: `client/src/components/HeroSection.tsx` (update image URL)

### Modify Skills
Edit: `client/src/components/SkillsSection.tsx`

### Update Projects
Edit: `client/src/components/ProjectsSection.tsx`

### Change Colors
Edit: `client/src/index.css` (CSS variables)
